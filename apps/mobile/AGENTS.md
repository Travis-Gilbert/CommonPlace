# Expo HAS CHANGED

Read the exact versioned docs at https://docs.expo.dev/versions/v57.0.0/ before writing any code.

## Recent product decisions

- The chat composer plus control opens the capability sheet for files, objects, plugins, skills, and web search. It never opens a second composer.
- Thread chrome keeps the back and new-thread actions, but does not render a separate "New thread" title bar.
