# Noun Project console glyphs

Canonical product glyphs enter the console through this directory. Filenames retain the Noun Project icon ID. Run `npm run icons:normalize` after adding an SVG; CI runs `gate:icons` to reject hard-coded paint.

These marks are already present elsewhere in this workspace and are supplied under the account holder's paid royalty-free plan, so product use does not require attribution. The account holder remains responsible for keeping that license coverage current.
